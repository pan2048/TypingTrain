inner class in Java

An inner class instance shares with an instance of the outer class. An inner class instance has access to all members of the outer class, even those marked private.
To instantiate an inner class form within the outer class.
<pre><code>
class Outer {
  private int i = 1;
  private class Inner {
    private void say() {
      System.out.println(i);
    }
  }
  private void create() {
    Inner inner = new Inner();
    inner.say();
  }

  public static void main(String[] args) {
    Outer outer = new Outer();
    outer.create();
  }
}
</code></pre>
To create an inner class object from outside the outer class instance code.
<pre><code>
class Outer {
  private int i = 1;
  private class Inner {
    private void say() {
      System.out.println(i);
    }
  }

  public static void main(String[] args) {
    Outer outer = new Outer();
    Outer.Inner inner = outer.new Inner();
    inner.say();
  }
}
</code></pre>
